What are Power Conservative Routes?

Power conservative routes are navigation routes with the power consumed by an electric vehicle battery factored in. Using google maps and live data from an electric vehicle driving around Logan, UT, the routes are both utility and experiment. This project uses power consumption data at various GPS coordinates to determine which route will use the least power.

UPDATE: Because the live data is not yet available, I have opted to model the battery usage in watt-hours/mile for a Tesla Model S (Data is easier to find for this vehicle, apparently they're popular or something.) I opted to use this model of the battery usage: https://my.teslamotors.com/no_NO/forum/forums/whmile-equation

Using the Google Maps API

The google maps API is an essential piece of making this project possible. All of the route data and ability to embed maps into the page comes from this API. Once start and end locations are entered into the page, the API is called and an array of possible routes are returned. The goal is to calculate the watt-hours per route based on the battery usage model.

Why Power Conservative Routes?

Power conservative routes are the next step in the electric vehicle revolution. One of the most prominent shortcomings of driving an electric vehicle is being limited by places the vehicle can be recharged. This makes getting as far as possible on each vehicle charge a priority. Besides, it saves energy! Isn't that the whole point of an electric vehicle?
